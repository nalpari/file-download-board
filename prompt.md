- nextjs + typescript + tailwindcss + vercel
- db는 postgresql
- ci/cd 는 docker 로 하고 docker compose로 실행한다.
- 파일을 업로드 하고 파일을 다운로드 할 수 있는 게시판을 구현한다.
- 파일이 이미지라면 상세보기에서 이미지를 미리보기 할 수 있도록 한다.
- 파일이 첨부 파일이라면 상세보기에서 미리보기 할 수 있도록 한다. 예를 들어 pdf 파일이라면 pdf를 이미지 미리보기 형태로 미리보기 할 수 있도록 한다.
- 파일 저장은 우선 로컬 파일 시스템을 사용한다.
